## DL_Project_Image_Caption_Generator

## Project Info
This Deep Learning project generates **automatic captions for images** using a **CNN + LSTM** architecture.  
A CNN extracts visual features from images, and an LSTM generates meaningful natural language descriptions.

---

## Requirements

1 - Python  
2 - VS Code / Jupyter Notebook  
3 - Git & GitHub  

---

## Libraries Used

1 - NumPy  
2 - Pandas  
3 - TensorFlow / Keras  
4 - NLTK  
5 - Flask  

---

## Dataset Info

- **Flickr8k Dataset**
- 8,000 images
- 5 captions per image

Dataset Link:  
https://www.kaggle.com/datasets/adityajn105/flickr8k

---

## Model Used

- **CNN (VGG16)** – Feature Extraction  
- **LSTM** – Caption Generation  

---

## Steps Used in This Project

1 - Data Preprocessing  
2 - Feature Extraction  
3 - Model Training  
4 - Caption Generation  
5 - Deployment using Flask  

---

## Deployment
The model is deployed using **Flask**, allowing users to upload an image and receive a generated caption.
